## 📌 Sobre o Projeto

O **Organo - Clube do Livro** é uma aplicação React desenvolvida para organizar e categorizar os livros lidos no *Clube do Livro com Prioli e Karnal* divididos por edições de anos (de 2020 a 2026). A plataforma permite cadastrar novos livros vinculando título, autor, imagem de capa e ano de edição correspondente.

### Tópicos abordados no desenvolvimento:
* **Conceitos de React:** Criação de componentes reutilizáveis e fluxo de dados unidirecional via `props`.
* **Gerenciamento de Estado:** Uso do hook `useState` para controle dinâmico dos livros cadastrados.
* **Manipulação de Listas:** Uso do método `.map()` e `.filter()` para renderização condicional dos times/anos.
* **Formulários:** Captura de eventos do usuário (`onSubmit`, `onChange`) e validação simples.

## 🚀 Como Executar o Projeto

Antes de começar, certifique-se de ter o **[Node.js](https://nodejs.org/)** (versão 16 ou superior) e o **npm** instalados na sua máquina.

### Passo a passo:

## 🚀 Como Executar o Projeto

1. **Clone este repositório:**
   git clone git@github.com:patyfil/clube-do-livro-prioli-karnal.git

2. **Acesse a pasta do projeto:**
   cd clube-do-livro-prioli-karnal

3. **Instale as dependências:**
   npm install

4. **Inicie o servidor de desenvolvimento:**
   npm start

5. **Acesse no navegador:**
   O projeto abrirá automaticamente no endereço http://localhost:3000.
